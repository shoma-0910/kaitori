## はじめに

このシステムは、買取催事（中古品買取イベント）を効率的に管理するためのツールです。

**主な機能：**
- 催事を開催する店舗の選定・登録
- イベントのスケジュール管理
- 売上データの分析
- AIによる店舗・地域の分析

---

## ログイン方法

1. アプリを開くとログイン画面が表示されます
2. **メールアドレス**と**パスワード**を入力
3. 「ログイン」ボタンをクリック

**新規登録の場合：**
1. 「新規登録」タブをクリック
2. メールアドレス、パスワード、名前を入力
3. 「登録」ボタンをクリック

---

## ダッシュボード

ログイン後に最初に表示される画面です。

**表示される情報：**
| 項目 | 説明 |
|------|------|
| 対象店舗数 | 登録されている店舗の数 |
| 予定催事件数 | これから開催予定のイベント数 |
| 総実績粗利 | これまでの利益の合計 |
| 総概算コスト | これまでのコストの合計 |

**使い方：**
- 画面上部で組織を切り替えられます（複数組織に所属している場合）
- 各数字をクリックすると詳細ページに移動します

## 店舗選定・予約

新しい催事場所を探すための機能です。

### 店舗を探す手順

1. 左メニューから「店舗選定・予約」をクリック
2. 地図上で催事を開催したい地域を表示
3. 「このエリアで検索」ボタンをクリック
4. 周辺のスーパーマーケットが地図上に表示されます

### 店舗の詳細を見る

1. 地図上のマーカー（ピン）をクリック
2. 店舗の情報が表示されます
   - 店舗名
   - 住所
   - 電話番号
   - 営業時間
   - 周辺の人口統計データ

### 店舗を登録する

1. 店舗情報の画面で「この店舗を登録」ボタンをクリック
2. 登録が完了すると「登録店舗一覧」に追加されます

### 予約リクエストを送る

1. 店舗情報の画面で「予約リクエスト」タブをクリック
2. 以下を入力：
   - 開催日（開始日・終了日）
   - 担当者名
   - 備考（任意）
3. 「予約リクエストを送信」ボタンをクリック
4. 予約代行者が承認すると、カレンダーにイベントが登録されます

## 登録店舗一覧

登録済みの店舗を管理する画面です。

### 店舗一覧を見る

1. 左メニューから「登録店舗」をクリック
2. 登録されている店舗がカード形式で表示されます

### 店舗の詳細を確認する

1. 店舗カードをクリック
2. 詳細画面が開きます

**確認できる情報：**
- 基本情報（住所、電話番号、営業時間）
- ランク（S/A/B/C - AIが自動判定）
- 売上履歴
- 周辺施設情報

### 売上を入力する

1. 店舗の詳細画面を開く
2. 「売上履歴」タブをクリック
3. 「売上を追加」ボタンをクリック
4. 以下を入力：
   - 売上金額
   - 買取点数
   - 日付
5. 「保存」ボタンをクリック

## カレンダー・スケジュール

イベントの予定を管理する画面です。

### カレンダーを見る

1. 左メニューから「スケジュール」をクリック
2. カレンダーが表示されます

**表示の切り替え：**
- 月表示 / 週表示 / 日表示 を切り替えられます

**色の意味：**
| 色 | 状態 |
|----|------|
| 青 | 予定（これから開催） |
| 緑 | 実施中 |
| グレー | 終了 |
| 点線グレー | 要請中（承認待ち） |

### 新しいイベントを作成する

1. カレンダー上の日付をクリック
2. 入力フォームが表示されます
3. 以下を入力：
   - 店舗（登録済み店舗から選択）
   - 開催日
   - 担当者名
   - 概算コスト（任意）
   - メモ（任意）
4. 「作成」ボタンをクリック

### イベントの詳細を確認・編集する

1. カレンダー上のイベントをクリック
2. 詳細画面が開きます
3. 必要に応じて情報を編集できます

### イベントを完了する（売上を入力）

1. 終了したイベントをクリック
2. 「売上を入力」をクリック
3. 以下を入力：
   - 売上金額
   - 買取点数
   - 実際のコスト
4. 「保存」ボタンをクリック

## 売上分析

売上データをグラフで確認できる画面です。

### 画面を開く

1. 左メニューから「売上分析」をクリック

### 確認できるデータ

**グラフ：**
- 店舗別売上ランキング（棒グラフ）
- 月別売上推移（折れ線グラフ）

**フィルター機能：**
- 期間で絞り込み（今週/今月/全期間など）
- 店舗で絞り込み

### 直接売上を入力する

イベントを通さずに売上を入力したい場合：

1. 「直接売上入力」タブをクリック
2. 店舗を選択
3. 売上金額、買取点数、日付を入力
4. 「登録」ボタンをクリック

---

## AI地域分析

AIが地域の買取ポテンシャルを分析する機能です。

### 使い方

1. 左メニューから「AI地域分析」をクリック
2. 都道府県を選択
3. 市区町村を選択
4. 「分析開始」ボタンをクリック

### 分析結果

AIが以下の観点から地域を評価します：
- 人口構成（高齢者・女性の割合）
- 平均所得
- 住居タイプ（持ち家率など）
- 買取イベントへの適性スコア

## AIおすすめ店舗

AIが店舗を自動でランク付けする機能です。

### AI推薦モード

1. 左メニューから「AIおすすめ店舗」をクリック
2. 「AIおすすめ」タブを選択
3. 地域を選択して「AI分析開始」をクリック
4. AIがS/A/B/Cランクで店舗を評価

**ランクの意味：**
| ランク | 意味 |
|--------|------|
| S | 最高評価（強くおすすめ） |
| A | 高評価（おすすめ） |
| B | 標準的 |
| C | 要検討 |

### 手動検索モード

1. 「手動検索」タブを選択
2. 以下の条件を設定：
   - 都道府県
   - 市区町村
   - 店舗タイプ
   - 検索半径
3. 「検索」ボタンをクリック

## 組織設定

組織やメンバーを管理する画面です（管理者のみ）。

### メンバーを追加する

1. 左メニューから「設定」をクリック
2. 「メンバー管理」セクションを開く
3. 「メンバーを追加」ボタンをクリック
4. メールアドレスと役割を入力
5. 「追加」ボタンをクリック

### メンバーの役割

| 役割 | できること |
|------|-----------|
| 管理者（admin） | すべての機能を使用可能 |
| メンバー（member） | 閲覧と基本的な操作のみ |

### 予約代行者を追加する

1. 「予約代行者管理」セクションを開く
2. 「予約代行者を追加」ボタンをクリック
3. 対象ユーザーを選択
4. 「追加」ボタンをクリック

## 予約代行（管理者向け）

予約リクエストを承認・拒否する画面です。

**対象：** 予約代行者として登録されたユーザー

### 予約リクエストを確認する

1. 左メニューから「予約管理」をクリック
2. 組織ごとに予約リクエストが表示されます

### 承認する

1. リクエストの「承認」ボタンをクリック
2. 確認画面で「承認する」をクリック
3. 自動的にイベントが作成され、申請者に通知が届きます

### 拒否する

1. リクエストの「拒否」ボタンをクリック
2. 拒否理由を入力（任意）
3. 「拒否する」ボタンをクリック
4. 申請者に通知が届きます

---

## 通知機能

システムからのお知らせを確認できます。

### 通知を確認する

1. 画面右上のベルマークをクリック
2. 未読の通知がある場合、数字が表示されます

### 通知の種類

| 通知タイプ | 内容 |
|-----------|------|
| 予約承認 | 予約リクエストが承認されました |
| 予約拒否 | 予約リクエストが拒否されました |

## よくある質問（FAQ）

### Q: パスワードを忘れました
A: ログイン画面の「パスワードを忘れた方」から再設定できます。

### Q: 予約リクエストが承認されたのにカレンダーに表示されない
A: ページを更新（リロード）してください。承認後は自動的にイベントが作成されます。

### Q: 組織を切り替えたい
A: ダッシュボード上部の組織選択ドロップダウンから切り替えられます。

### Q: スマートフォンでも使えますか？
A: はい、ブラウザから利用できます。一部の機能は画面サイズに応じて表示が変わります。
